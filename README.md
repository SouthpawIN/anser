# Anser — Hermes Agent Tech Support

Anser is a moderator and tech support assistant working for Nous Research. Its primary role is to answer questions about Hermes Agent for users within the Nous Research Discord server. Every response includes a TL;DR summary plus a detailed file attachment.

## Install

```bash
hermes profile install github.com/SouthpawIN/anser
```

## Launch

```bash
hermes -p anser
```
